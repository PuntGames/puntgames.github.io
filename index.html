PuntGames Collection

PuntGames is a collection of games freely downloadable, with no DRM, no sitelock, no restrictions, and no garbage. Submit your games that you want to add at puntgames AT proton DOT me. You can download the individual games as .zip archives that should work when opened in the browser, or just browse them here. (Some games hosted here don't work on file URLs, however.) All of the games hosted on this site are free games that are legally allowed to be hosted elsewhere. Making mirrors of the repository is both allowed and encouraged. If a link says Play + Download, press Ctrl+S on the page to save it locally.

You can also submit games here.
